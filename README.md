# Grokking Algorithms

This repository contains my studies, notes, and implementations based on the book  
**"Grokking Algorithms" by Aditya Bhargava**.

The goal is to deeply understand core algorithms through theory and hands-on practice.

---

## About the Book

is a beginner-friendly guide that explains algorithms using visual examples and practical scenarios.

It covers essential topics like:

- Searching & Sorting  
- Recursion  
- Hash Tables  
- Graph Algorithms (BFS, Dijkstra)  
- Greedy Algorithms  
- Dynamic Programming  

The book focuses on intuition and real-world understanding rather than heavy theory :contentReference[oaicite:1]{index=1}.

## Objective

- Understand how algorithms work in practice  
- Build intuition for problem solving  
- Implement each algorithm from scratch  
- Prepare for technical interviews
